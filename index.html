let和const

let
1、声明的变量只有在当前作用域中有效
2、在同一个作用域不能重复声明,否则会报错
{
let a=10;
let a=20;
}
//Uncaught SyntaxError: Identifier 'a' has already been declared
3、不会变量提升，var会在预编译环节变量的声明会提升至顶部
console.log(a)
let a = 10;
//Uncaught ReferenceError: a is not defined
4、在全局作用域创建a和window.a不相同，var a会修改window.a的值

const
1、声明的是常量，不能修改
const A= 10;
A=20;
//Uncaught TypeError: Assignment to constant variable.
2、声明的常量只有在当前作用域中有效
3、在同一个作用域不能重复声明,否则会报错
4、声明的常量必须初始化
const A
//Uncaught SyntaxError: Missing initializer in const declaration
5、不会变量提升
6、不允许修改绑定，但允许修改值
对象
const person = {
name:”xiaoming”
}
person.name = “zhangsan”
console.log(person)
//Object {name: “zhangsan”}

const person = {
            name:"xiaoming"
        }
        person = {
            name:"zhangsan"
        }
        console.log(person)    
//报错

数组
const a = [1,2,3]
a[0] = 3;
console.log(a)
//[3, 2, 3]

const a = [1,2,3]
    a = [3,2,3]
    console.log(a)
//报错

7、在全局作用域创建a和window.a不相同
