let和const

这就是一个简介

let
1、声明的变量只有在当前作用域中有效
2、在同一个作用域不能重复声明,否则会报错


3、不会变量提升，var会在预编译环节变量的声明会提升至顶部

4、在全局作用域创建a和window.a不相同，var a会修改window.a的值

const
1、声明的是常量，不能修改

2、声明的常量只有在当前作用域中有效
3、在同一个作用域不能重复声明,否则会报错
4、声明的常量必须初始化

5、不会变量提升
6、不允许修改绑定，但允许修改值
对象

数组


const a = [1,2,3]
a = [3,2,3]
console.log(a)
//报错
7、在全局作用域创建a和window.a不相同
